# Getting Started

## Prerequisite
To use **Minomax**, you will need the following prerequisites:

- **Node.js (v16.x or later)** installed on your system. You can download it from
[https://nodejs.org/en/download](https://nodejs.org/en/download).
- **NPM** (Node Package Manager) or **YARN** should be installed along with Node.js.

Once you have these prerequisites in place, you can follow the installation steps in the project's documentation
to set up and run the project on your local machine.

## Installation

```sh npm2yarn
npm install -D @cresteem/minomax
```
