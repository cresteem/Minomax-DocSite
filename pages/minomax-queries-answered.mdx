# Frequently Asked Questions

<details>
    <summary>What is Minomax?</summary>
    <p>Minomax is a tool designed to optimize web projects by compressing images, videos, and web documents, as well as generating image sets for different devices. It simplifies the optimization process with minimal configuration.</p>
</details>

<details>
    <summary>How do I install Minomax?</summary>
    <p>You can install Minomax globally using npm by running the command: `npm install -g @cresteem/minomax`</p>
</details>

<details>
    <summary>What are the core features of Minomax?</summary>
    <p>Minomax offers four core features: image compression, video compression, image set generation for different devices, and web document compression.</p>
</details>

<details>
    <summary>How do I run the Minomax process?</summary>
    <p>You can run the entire Minomax process using the command: `minomax make` with various options to customize the output format, codec, encoding level, destination path, and ignore patterns.</p>
</details>

<details>
    <summary>How can I compress images using Minomax?</summary>
    <p>You can compress images by using the command: `minomax image` with options for specifying path patterns, output format, destination path, and ignore patterns.</p>
</details>

<details>
    <summary>What video formats are supported by Minomax?</summary>
    <p>Minomax supports three video codecs: `wav1` (WebM format with AV1 codec), `mav1` (MP4 container with AV1 codec), and `mx265` (Nvidia Encoder of X265). For the most compressed output, `wav1` is recommended, while for faster compression, the Nvidia Encoder X265 is preferable.</p>
</details>

<details>
    <summary>Can Minomax minify JavaScript, CSS, and HTML files?</summary>
    <p>Yes, Minomax can minify web documents such as JavaScript, CSS, and HTML files using the command: `minomax minify`.</p>
</details>

<details>
    <summary>What is image set generation in Minomax?</summary>
    <p>Image set generation in Minomax creates multiple versions of images for different screen sizes to deliver dynamic content, optimizing the display for various devices.</p>
</details>

<details>
    <summary>How do I generate image sets using Minomax?</summary>
    <p>You can generate image sets by using the command: `minomax genset` with options for specifying path patterns, destination path, and ignore patterns.</p>
</details>

<details>
    <summary>What options are available for the `minomax make` command?</summary>
    <p>The `minomax make` command includes options for specifying the image output format, video codec type, video encoding level, destination base path, and ignore path patterns.</p>
</details>

<details>
    <summary>Does Minomax support CLI and API usage?</summary>
    <p>Yes, Minomax provides two ways to work with it: CLI (Command Line Interface) and API (Application Programming Interface).</p>
</details>

<details>
    <summary>Can Minomax handle video compression efficiently?</summary>
    <p>Yes, Minomax supports efficient video compression, achieving a compression ratio of 93.16%.</p>
</details>

<details>
    <summary>Does Minomax automatically convert image tags to modern picture tags?</summary>
    <p>Yes, Minomax automatically converts image tags to modern picture tags with media queries for different devices, catering different images with image sets.</p>
</details>

<details>
    <summary>How does Minomax handle video thumbnails?</summary>
    <p>Minomax generates video thumbnails and includes them automatically inside the HTML, similar to how it handles picture tags from image tags.</p>
</details>

<details>
    <summary>Where can I find more information or support for Minomax?</summary>
    <p>For more detailed information, advanced usage, and troubleshooting, please refer to the full documentation or contact our support team.</p>
</details>

<details>
    <summary>What are the default encoding levels for video compression in Minomax?</summary>
    <p>Minomax offers three encoding levels for video compression: 1 (lowest), 2 (medium), and 3 (highest). You can specify the desired level using the `-e` option.</p>
</details>

<details>
    <summary>What is the best video codec type for maximum compression?</summary>
    <p>For the most compressed output, `wav1` (WebM format with AV1 codec) is recommended.</p>
</details>

<details>
    <summary>What is the best video codec type for faster compression?</summary>
    <p>For faster compression, the Nvidia Encoder X265 (`mx265`) is preferable.</p>
</details>

<details>
    <summary>Can Minomax be used for both personal and commercial projects?</summary>
    <p>Yes, Minomax can be used for both personal and commercial projects, providing a versatile tool for web optimization.</p>
</details>

<details>
    <summary>How does Minomax handle different image formats?</summary>
    <p>Minomax allows you to specify the image output format (jpg, avif, or webp) using the <code>-f</code> option in the commands. The default format is webp.</p>
</details>

<details>
    <summary>What is the configuration process for Minomax?</summary>
    <p>Minomax simplifies the configuration process with minimal setup required. Users can easily specify options through the command line interface, and advanced configurations can be managed via a JSON configuration file.</p>
</details>

<details>
    <summary>How do I exclude specific files or directories from Minomax processing?</summary>
    <p>You can exclude specific files or directories by using the <code>--ignore</code> option followed by the path patterns you wish to ignore. This option is available for all Minomax commands.</p>
</details>

<details>
    <summary>Does Minomax support batch processing?</summary>
    <p>Yes, Minomax supports batch processing by allowing you to specify path patterns for multiple files at once. This feature is available for image compression, video compression, web document minification, and image set generation.</p>
</details>
