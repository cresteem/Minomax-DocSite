# Frequently Asked Questions

<details>
  <summary>What is Minomax?</summary>
  <p>
    Minomax is a tool designed to optimize web projects by compressing images,
    videos, and web documents, as well as generating image sets for different
    devices. It simplifies the optimization process with minimal configuration.
  </p>
</details>

<details>
  <summary>How do I use the `prod` command in Minomax CLI?</summary>
  <p>
    The `prod` command runs the Minomax optimization process for images and
    videos. You can specify options like image format (`jpg`, `avif`, `webp`),
    video codec (`wav1`, `mav1`, `mx265`), and video encoding levels (1 to 3) to
    tailor the optimization.
    <br />
    <strong>Example:</strong>
    <pre>
      minomax prod --format webp --codec mav1 --encode 2 --ignore "node_modules"
    </pre>
  </p>
</details>

<details>
  <summary>How can I compress images with Minomax CLI?</summary>
  <p>
    Use the `image` command to compress images. Specify path patterns to locate
    the image files, set the output format, and choose the destination path for
    optimized images.
    <br />
    <strong>Example:</strong>
    <pre>
      minomax image --patterns "src/images/**/*.jpg" --format webp --dest
      "./optimized_images"
    </pre>
  </p>
</details>

<details>
  <summary>What video codecs are supported by Minomax CLI?</summary>
  <p>
    Minomax CLI supports three video codecs: <code>wav1</code>,{" "}
    <code>mav1</code>, and <code>mx265</code>. You can specify the desired codec
    using the <code>--codec</code> option when using the <code>video</code>{" "}
    command.
  </p>
</details>

<details>
  <summary>How do I compress videos using Minomax CLI?</summary>
  <p>
    The `video` command allows you to compress video files. You can choose a
    codec, set the encoding level, and define the destination for compressed
    videos.
    <br />
    <strong>Example:</strong>
    <pre>
      minomax video --patterns "src/videos/**/*.mp4" --codec mav1 --encode 2
      --dest "./optimized_videos"
    </pre>
  </p>
</details>

<details>
  <summary>What is the purpose of the `minify` command?</summary>
  <p>
    The `minify` command is used to minify web documents such as HTML, JS, and
    CSS files. It reduces the file size by removing unnecessary characters
    without changing the functionality of the code, leading to improved page
    load speeds.
    <br />
    <strong>Example:</strong>
    <pre>
      minomax minify --patterns "src/**/*.html" --dest "./dist" --ignore
      "node_modules"
    </pre>
  </p>
</details>

<details>
  <summary>
    Can I exclude specific files or directories from processing in Minomax CLI?
  </summary>
  <p>
    Yes, you can use the <code>--ignore</code> option followed by path patterns
    to exclude specific files or directories from processing.
    <br />
    <strong>Example:</strong>
    <pre>minomax prod --ignore "node_modules, dist"</pre>
  </p>
</details>

<details>
  <summary>How do I generate responsive image sets with Minomax CLI?</summary>
  <p>
    Use the `genset` command to generate image sets for responsive design. This
    will create multiple image sizes for different screen resolutions.
    <br />
    <strong>Example:</strong>
    <pre>
      minomax genset --patterns "src/images/**/*.jpg" --dest
      "./responsive_images"
    </pre>
  </p>
</details>

<details>
  <summary>What does the `init` command do in Minomax CLI?</summary>
  <p>
    The `init` command initializes a configuration template for Minomax. It sets
    up the necessary configuration files and prepares the project for
    optimization.
    <br />
    <strong>Example:</strong>
    <pre>minomax init</pre>
  </p>
</details>

<details>
  <summary>
    Can I configure Minomax CLI to optimize only certain file types?
  </summary>
  <p>
    Yes, you can use the <code>--patterns</code> option to specify the types of
    files (e.g., images, videos, web documents) you want to process. For
    example, using <code>"*.jpg"</code> to compress only JPEG images.
  </p>
</details>

<details>
  <summary>
    What is the difference between the `prod` and `image` commands?
  </summary>
  <p>
    The `prod` command optimizes both images and videos together, while the
    `image` command focuses solely on compressing images. The `prod` command is
    a comprehensive optimization option for both types of media.
  </p>
</details>

<details>
  <summary>Can I run Minomax CLI on any operating system?</summary>
  <p>
    Yes, Minomax CLI is built with cross-platform compatibility. It should work
    on Linux, macOS, and Windows, as long as Node.js is installed.
  </p>
</details>

  </p>
</details>
